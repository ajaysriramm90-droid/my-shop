# About AjayShop

AjayShop is a modern eCommerce web application developed using TypeScript, React, and GraphQL.

## Why I Built This
This project was built to demonstrate full-stack development skills including frontend with React, backend with Node.js, database management with PostgreSQL, and API design with GraphQL.

## What I Learned
- TypeScript for type-safe development
- GraphQL API design and implementation
- PostgreSQL database management
- Docker containerization
- React component architecture
- Node.js backend development
