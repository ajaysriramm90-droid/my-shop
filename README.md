# AjayShop 🛍️

A modern full-stack eCommerce platform built with TypeScript, React, GraphQL and PostgreSQL.

## 🚀 Tech Stack
- TypeScript
- React
- GraphQL
- PostgreSQL
- Node.js
- Docker

## ✨ Features
- Product catalog and management
- Shopping cart functionality
- Admin panel with dashboard
- Order management system
- Customer authentication
- Multi-language support

## 🛠️ Installation

Step 1 - Clone the project:
```
git clone https://github.com/ajaysriramm90-droid/my-shop.git
```

Step 2 - Install dependencies:
```
npm install
```
createdb -U postgres evershop



Step 3 - Setup the database:
```
npm run setup
```

Step 4 - Start development server:
```
npm run dev
```

Step 5 - Open browser at:
```
http://localhost:3000
```

## 📁 Project Structure
- packages/evershop - Core platform code
- packages/postgres-query-builder - Database query builder
- translations/ - Multi-language support files
- seed/ - Sample product images and data
